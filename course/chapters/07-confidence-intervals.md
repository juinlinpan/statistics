# 第 7 章：信賴區間

## 為什麼要學這一章

民調說某候選人的支持率是 52%，這個 52% 並不是母體支持率本身，而是抽到這一批受訪者後得到的估計。若換一批人，答案通常會改變。只報一個數字，會把這種抽樣不確定性藏起來。

**信賴區間（confidence interval, CI）**用「合理估計的範圍」表達不確定性。本章學完後，你不只會算區間，也能判斷區間何時可信、何時不能使用，以及「95% 信賴」究竟在說什麼。

## 先備檢查

開始前，請確認你能區分：

- **母體參數（parameter）**：固定但通常未知，例如所有選民的真實支持比例 $p$。
- **樣本統計量（statistic）**：由樣本算出、會隨抽樣改變，例如樣本比例 \hat p。
- **標準差（standard deviation, SD）**：描述個別觀測值的分散。
- **標準誤（standard error, SE）**：描述統計量在重複抽樣下的分散。
- **抽樣分配**與中央極限定理（central limit theorem, CLT）的基本概念。

若「資料有多分散」與「估計值有多不穩定」仍容易混淆，先回顧第 5 章的標準差與標準誤。

## 學習目標

完成本章後，你能：

1. 用重複抽樣的觀點正確解讀信賴區間。
2. 從「估計值 ± 臨界值 × 標準誤」建立 CLT 型信賴區間。
3. 計算平均數與比例的常見大樣本信賴區間，並檢查使用條件。
4. 用 bootstrap 重抽樣估計標準誤，並說明它能與不能解決的問題。
5. 解釋信心水準、涵蓋率、誤差範圍與樣本數之間的關係。

## 1. 從點估計走向區間估計

假設隨機抽取 400 位選民，其中 208 位支持某候選人。樣本比例是 $\hat p=208/400=0.52$。這個 **點估計（point estimate）**是目前對未知母體比例 $p$ 的最佳單一猜測，但沒有呈現精確程度。

區間估計的核心想法是：若已知估計值在重複抽樣時通常偏離參數多遠，就能在點估計兩側留出合理的緩衝範圍。

### 信賴區間是隨機程序的產物

想像從同一母體重複抽樣 100 次，每次都用同一套規則建立一個 95% 信賴區間。長期而言，大約 95 個區間會蓋住固定的真實參數，約 5 個不會。每次抽到的樣本不同，所以區間端點也不同；母體參數並沒有跟著移動。

**95% 信賴區間的正確解讀**是：建立這個區間的程序，在假設成立且反覆使用時，長期涵蓋真實參數的比例為 95%（或在近似方法下約為 95%）。

在標準頻率學派（frequentist）解讀下，資料一旦觀察完成，這個已算出的區間要嘛蓋住參數、要嘛沒有。不能說「參數有 95% 機率落在這個已實現的區間內」。95% 描述的是方法的長期表現，不是固定參數的事後機率。

### 非例：區間不是資料的容器

平均反應時間的 95% 信賴區間若為 $[230,250]$ 毫秒，不代表 95% 的個別反應時間位於 230 到 250 毫秒之間。信賴區間估計的是**母體平均反應時間**；個別資料的範圍通常寬得多。

### 小測一下

某研究報告平均睡眠時間的 95% 信賴區間為 6.8 到 7.2 小時。下列哪一句正確？

> 這套建構區間的方法在重複抽樣下，約有 95% 的區間會包含母體平均睡眠時間。

「95% 的人睡 6.8 到 7.2 小時」與「母體平均值有 95% 機率在此區間」都不是標準頻率學派的正確解讀。

## 2. CLT 型信賴區間的共同結構

若估計量的抽樣分配近似常態，約 95% 的估計值會落在真實參數上下約 1.96 個標準誤內。把關係反過來看，就得到以觀察到的估計值為中心的區間。

<a id="formula-ci-estimate-critical-se"></a>

### 一般 CLT 型信賴區間

\[
\text{點估計}\ \pm\ \text{臨界值}\times \text{SE(點估計)}
\]

| 符號或部分 | 意義 | 單位 |
|---|---|---|
| 點估計 | 用樣本統計量估計目標參數 | 與參數相同 |
| 臨界值 | 由信心水準與參考分配決定的分位數 | 無單位 |
| \(\text{SE(點估計)}\) | 點估計的抽樣標準差；實務上常以資料估計 | 與參數相同 |
| 臨界值 \(\times\) SE | **誤差範圍（margin of error, MOE）** | 與參數相同 |

常態參考分配下，雙側 90%、95%、99% 信賴區間的臨界值約為 1.645、1.96、2.576。臨界值沒有單位，因此區間端點和參數維持相同單位。

**使用時機：**估計量的抽樣分配近似常態、觀測值的抽樣機制合理，且標準誤能可靠估計時。

**不要使用的情況：**樣本有嚴重選擇偏誤、觀測值高度相依卻當作獨立、樣本太小且分配高度偏斜，或參數位於邊界使常態近似失真。增加樣本數可減少隨機誤差，卻不會自動修正偏誤。

這個公式是一個結構，不表示任何問題都應固定使用 1.96。小樣本平均數常需用 $t$ 臨界值；抽樣分配不近似常態時，可能需要其他理論方法或重抽樣區間。

### 為什麼可以把不等式反過來？

若估計量 \(\hat\theta\) 的抽樣分配以參數 \(\theta\) 為中心且近似常態，則

\[
P\left(-z^* \leq \frac{\hat\theta-\theta}{\operatorname{SE}(\hat\theta)} \leq z^*\right)\approx C,
\]

其中 $C$ 是信心水準，$z^*$ 是使常態曲線中央面積為 $C$ 的臨界值。整理不等式後得到

\[
P\left(\hat\theta-z^*\operatorname{SE}(\hat\theta)
\leq \theta \leq
\hat\theta+z^*\operatorname{SE}(\hat\theta)\right)\approx C.
\]

機率陳述針對抽樣前會改變的端點。抽樣後，端點成為固定數值，便改用「此程序的信心水準為 $C$」來表達。

## 3. 母體平均數的大樣本區間

設 $X_1,\ldots,X_n$ 是近似獨立、來自同一母體的觀測值。樣本平均數為 $\bar x$，樣本標準差為 $s$。當樣本數夠大、極端值不致主導結果時，CLT 讓 $\bar X$ 的抽樣分配近似常態，且其標準誤可用 $s/\sqrt n$ 估計。

<a id="formula-ci-mean-large-sample"></a>

### 母體平均數的大樣本信賴區間

\[
\bar x \pm z^*\frac{s}{\sqrt n}
\]

| 符號 | 意義 | 單位 |
|---|---|---|
| \(\bar x\) | 樣本平均數，估計母體平均數 \(\mu\) | 與觀測值相同 |
| $s$ | 樣本標準差，估計母體標準差 $\sigma$ | 與觀測值相同 |
| $n$ | 樣本中的獨立觀測數 | 無單位（筆數） |
| $z^*$ | 常態臨界值 | 無單位 |

**假設與限制：**樣本應來自隨機抽樣或可合理代表目標母體的設計；觀測值彼此獨立或相依性已被正確處理；$n$ 足以讓平均數的抽樣分配近似常態。若資料非常偏斜或有強烈離群值，需要更大的 $n$，不能只靠一條固定的「$n\ge 30$」規則。

**何時不用：**小樣本且母體標準差未知時，若資料近似常態，通常應考慮 $t$ 區間而不是直接使用 $z^*$。配對或群聚資料也不能把所有觀測值當作獨立樣本套入此式。

### 例：平均客服等待時間

隨機抽取 $n=100$ 通電話，平均等待時間 $\bar x=8.4$ 分鐘，樣本標準差 $s=3.0$ 分鐘。95% 大樣本區間為

\[
8.4 \pm 1.96\frac{3.0}{\sqrt{100}}
=8.4\pm0.588,
\]

也就是約 $[7.81,8.99]$ 分鐘。誤差範圍與平均數的單位都是分鐘，這是基本的量綱檢查。

合理解讀是：若抽樣設計與近似條件成立，這套方法建立的區間長期約有 95% 會包含母體平均等待時間。它不保證本次區間必然正確，也不表示 95% 的電話等待時間落在其中。

### 小測一下

若樣本標準差近似不變，把獨立樣本數從 100 增至 400，標準誤會變成原來多少？

> \(\sqrt{100/400}=1/2\)，因此標準誤與誤差範圍約減半，而不是變成四分之一。

## 4. 母體比例的大樣本區間

二元結果可記為成功 $1$、失敗 $0$。若 $X$ 是 $n$ 次近似獨立試驗中的成功次數，樣本比例 $\hat p=X/n$ 的標準誤估計為 $\sqrt{\hat p(1-\hat p)/n}$。

<a id="formula-ci-proportion-wald"></a>

### 母體比例的常態近似區間

\[
\hat p \pm z^*\sqrt{\frac{\hat p(1-\hat p)}{n}}
\]

| 符號 | 意義 | 單位 |
|---|---|---|
| $\hat p$ | 樣本成功比例 $X/n$，估計母體比例 $p$ | 無單位，可寫成百分比 |
| $X$ | 樣本中的成功次數 | 次數 |
| $n$ | 近似獨立試驗數 | 次數 |
| $z^*$ | 常態臨界值 | 無單位 |

**使用時機：**二元結果、樣本具有代表性、各次結果近似獨立，而且成功與失敗的觀察數都足以支持常態近似。常用的初步檢查是 $n\hat p\ge10$ 且 $n(1-\hat p)\ge10$，但這只是經驗準則，不是保證。

**何時不用：**事件很稀少、樣本很小或 \(\hat p\) 接近 0 或 1 時，這個又稱 Wald 區間的近似可能很差，甚至產生小於 0 或大於 1 的端點。此時應考慮更合適的二項比例區間方法，而不是把不合理端點硬截成 0 或 1。

### 例：支持比例

在 $n=400$ 人中有 $X=208$ 人表示支持，因此 $\hat p=0.52$。成功與失敗數為 208 與 192，常態近似檢查沒有顯著警訊。95% 區間為

\[
0.52\pm1.96\sqrt{\frac{0.52(0.48)}{400}}
\approx0.52\pm0.049,
\]

即約 $[0.471,0.569]$，也可寫成 47.1% 到 56.9%。這只反映此模型下的抽樣誤差；未回覆偏誤、問題措辭或抽樣框遺漏等系統性問題不會出現在這個誤差範圍裡。

### 非例：方便樣本不因人多而變隨機

在社群帳號上讓追蹤者自願投票，即使有十萬份回覆，也不能僅靠上述公式推論所有選民。大 $n$ 會讓公式算出的 SE 很小，卻無法修正自我選擇造成的偏誤。

## 5. 用 bootstrap 估計標準誤

有些統計量的理論標準誤不容易推導，例如中位數或複雜模型的估計值。**bootstrap（自助重抽樣法）**把原樣本視為母體的近似，從 $n$ 筆原資料中「放回抽樣」$n$ 次，算出一個 bootstrap 統計量；重複很多次後，這些統計量的分散可近似原估計量的抽樣分散。

流程如下：

1. 原樣本有 $n$ 筆資料。
2. 從原樣本放回抽取 $n$ 筆，形成一個 bootstrap 樣本。
3. 計算同一個統計量，例如平均數、中位數或迴歸斜率。
4. 重複步驟 2–3 共 $B$ 次，得到 $\hat\theta^{*1},\ldots,\hat\theta^{*B}$。
5. 用這 $B$ 個 bootstrap 統計量的樣本標準差估計 SE。

<a id="formula-bootstrap-standard-error"></a>

### Bootstrap 標準誤

\[
\widehat{\operatorname{SE}}_{\text{boot}}(\hat\theta)
=
\sqrt{\frac{1}{B-1}
\sum_{b=1}^{B}\left(\hat\theta^{*b}-\bar{\hat\theta}^{*}\right)^2},
\qquad
\bar{\hat\theta}^{*}=\frac{1}{B}\sum_{b=1}^{B}\hat\theta^{*b}.
\]

| 符號 | 意義 | 單位 |
|---|---|---|
| \(\hat\theta\) | 原樣本的統計量 | 與目標參數相同 |
| $\hat\theta^{*b}$ | 第 $b$ 個 bootstrap 樣本算出的統計量 | 與目標參數相同 |
| $\bar{\hat\theta}^{*}$ | $B$ 個 bootstrap 統計量的平均 | 與目標參數相同 |
| $B$ | bootstrap 重複次數 | 無單位 |

**關鍵條件：**原樣本必須能合理代表母體，且重抽樣方式要保留原研究設計的相依結構。獨立資料可逐筆重抽；配對、群聚或時間相依資料不能不加思索地逐筆重抽。

**能做什麼：**近似某統計量因抽樣而產生的變動，尤其適合解析 SE 難算的情況。

**不能做什麼：**bootstrap 不會創造原樣本沒有的資訊，也不會修正便利抽樣、未回覆或測量偏誤。若原樣本很小、極端值主導統計量，或統計量位於參數空間邊界，bootstrap 近似也可能不穩定。

若 bootstrap 分配近似對稱、單峰且偏誤不大，可把 bootstrap SE 放入一般結構：

\[
\hat\theta\pm z^*\widehat{\operatorname{SE}}_{\text{boot}}(\hat\theta).
\]

這不是所有 bootstrap 區間的通用配方。當分配明顯偏斜時，單純的「估計值 ± 1.96 SE」可能不合適；第 9 章會介紹以重抽樣分布直接建立區間的其他方法。

### 小測一下

為什麼 bootstrap 樣本要放回抽樣？

> 放回抽樣使同一筆原資料可出現多次、也可能完全沒出現，從而模擬重新從母體取得樣本時統計量的變化。若不放回且每次都抽滿 $n$ 筆，每個重抽樣只會是原資料的重新排序，統計量通常不會改變。

## 6. 信心水準、涵蓋率與誤差範圍

**信心水準（confidence level）**是方法設計的目標長期涵蓋比例，例如 95%。**涵蓋率（coverage）**則是某方法在指定資料生成條件下，實際涵蓋真實參數的長期比例。若近似條件不好，名義 95% 的方法實際涵蓋率可能不是 95%。

<a id="formula-margin-of-error"></a>

### 誤差範圍

\[
\operatorname{MOE}=\text{臨界值}\times \operatorname{SE}(\text{點估計}).
\]

因此，對同一批資料與同一方法：

- 提高信心水準會提高臨界值，使區間變寬。
- 增加獨立樣本數通常會降低 SE，使區間變窄。
- 資料變異越大，SE 通常越大，區間越寬。
- 在常見的獨立抽樣平均數與比例問題中，SE 約按 $1/\sqrt n$ 縮小；要把誤差範圍減半，樣本數約需變成四倍。

**寬不一定代表方法差，窄也不一定代表可信。**寬區間誠實呈現資訊有限；有偏誤的龐大樣本可能給出非常窄、卻系統性偏離真值的區間。

### 實務重要性與區間

區間提供一組與資料相容的參數值，可用來判斷效果的可能大小。若某改善方案的平均節省時間區間是 $[0.1,0.3]$ 秒，即使整段都大於 0，實務價值仍可能很小。反之，區間若很寬並跨過 0，可能表示資料同時容許有益與有害的效果；此時重點是資訊不足，而不是證明「沒有效果」。

信賴區間描述抽樣不確定性，不等於因果結論。觀察性研究即使區間很窄，仍可能受混雜因子影響。

## 章末整合：建立區間前的決策順序

遇到信賴區間問題時，依序問：

1. **目標參數是什麼？**平均數、比例、中位數，或其他量？
2. **點估計是什麼？**它和參數是否使用同一尺度與單位？
3. **資料如何產生？**樣本是否具代表性？觀測值是否獨立、配對或群聚？
4. **抽樣分配可否近似常態？**若不可，是否有合適的理論方法或重抽樣方法？
5. **SE 如何取得？**使用理論公式、估計公式，還是 bootstrap？
6. **臨界值與信心水準是否匹配？**
7. **解讀是否只談目標參數與抽樣不確定性？**避免把區間說成個別資料範圍、參數的事後機率或偏誤保證。

## 公式與術語速查

| 項目 | 核心內容 | 決策提醒 |
|---|---|---|
| 點估計 | 用一個樣本統計量估計參數 | 不呈現不確定性 |
| 一般 CLT 型 CI | 點估計 \(\pm\) 臨界值 \(\times\) SE | 需近似常態且 SE 可靠 |
| 平均數大樣本 CI | $\bar x\pm z^*s/\sqrt n$ | 小樣本常需考慮 $t$ 方法 |
| 比例常態近似 CI | \(\hat p\pm z^*\sqrt{\hat p(1-\hat p)/n}\) | 稀少事件或邊界附近不可靠 |
| Bootstrap SE | bootstrap 統計量的樣本標準差 | 重抽樣單位須符合研究設計 |
| 誤差範圍 | 臨界值 \(\times\) SE | 是區間半寬，不包含所有偏誤 |
| 信心水準 | 方法的目標長期涵蓋比例 | 不是固定參數的事後機率 |
| 涵蓋率 | 方法實際蓋住真值的長期比例 | 可能因近似失準而偏離名義水準 |

## 累積自我檢查

1. 一個 95% 信賴區間沒有包含真實參數，是否與「95% 信賴」矛盾？為什麼？
2. 在其他條件近似不變時，把樣本數加倍，誤差範圍會減半嗎？
3. 一份自願填答的網路調查有一百萬筆資料，極窄的區間是否保證代表全體人口？
4. 什麼情況下比例的常態近似區間可能產生誤導？
5. Bootstrap 為何不能修復原始樣本的選擇偏誤？

### 參考答案

1. 不矛盾。95% 方法長期仍容許約 5% 的區間未涵蓋真值，無法從單次結果辨認本次是否屬於那一部分。
2. 不會。SE 通常按 $1/\sqrt n$ 改變；樣本數加倍後，誤差範圍約乘上 $1/\sqrt2\approx0.707$。
3. 不保證。大樣本降低隨機誤差，不能消除自我選擇與涵蓋不足等系統性偏誤。
4. 成功或失敗次數太少、\(\hat p\) 靠近 0 或 1，或觀測不獨立時。
5. Bootstrap 只重複使用原樣本中的資訊；若原樣本系統性漏掉某群體，重抽樣也不會把該群體補回來。
