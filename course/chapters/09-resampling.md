# 第 9 章：重抽樣

## 為什麼要學這一章

假設你有 40 位顧客的等待時間，想知道「樣本中位數有多不穩定」。平均數的標準誤有熟悉的公式，但中位數呢？或者你的統計量是兩個分位數之差、迴歸斜率，甚至一段自訂的計算流程，又該怎麼辦？

**重抽樣(resampling)** 的核心想法是：讓電腦反覆模擬「重新取得資料」，觀察統計量會如何變動。它不是憑空創造新資訊，而是把手上的樣本或假設的機率模型當作近似世界，重複做原本很難用紙筆完成的機率計算。

本章會把兩種容易混淆的模擬分開：

- **Monte Carlo 模擬** ：從已知或指定的機率模型抽樣，近似某個機率或期望值。
- **bootstrap** ：用樣本估計未知的母體分配，再從這個估計分配重抽樣，評估統計量的不確定性。

重抽樣很有彈性，但不是「資料少也能變多」的魔法。原始樣本的偏誤、缺乏代表性或相依結構若沒有被正確建模，會被重複帶進每一次重抽樣。

## 先備知識檢查

閱讀前，請確認你能回答：

1. **母數(parameter)** 與 **統計量(statistic)** 有何不同？例如母體平均數 $\mu$ 與樣本平均數 $\bar{x}$。
2. **抽樣分配(sampling distribution)** 描述的是什麼重複過程？
3. 標準誤為何衡量統計量在重複抽樣下的變動，而不是個別觀測值的離散？
4. 95% 信賴區間的長期覆蓋率如何解讀？
5. 簡單線性迴歸中的 fitted value $\hat y_i$ 與 residual $e_i=y_i-\hat y_i$ 各代表什麼？

若第 1–4 題不熟，先回顧第 5 章與第 7 章；迴歸符號可回顧第 6 章。

## 學習目標

完成本章後，你將能：

- 判斷一個問題是在估計 Monte Carlo 誤差，還是在估計原始統計量的抽樣誤差。
- 用大數法則說明為何增加模擬次數能穩定模擬結果，以及它不能修正哪些誤差。
- 用 plug-in principle 建立非參數與參數式 bootstrap 的近似世界。
- 設計大小正確的 bootstrap 重抽樣，並由重抽樣結果估計偏誤與標準誤。
- 建構並正確解讀 percentile bootstrap 信賴區間。
- 根據迴歸問題的資料生成假設，選擇 pairs bootstrap 或 residual bootstrap。
- 辨認重抽樣失效或需要調整的情況，包括小樣本、離群值、相依資料與邊界參數。

## 1. 用模擬取代難算的機率

### 1.1 先把問題改寫成重複實驗

想知道一枚公平硬幣擲 20 次，至少出現 14 次正面的機率。你可以列出二項分配公式；也可以把「擲 20 次並判斷是否至少 14 次正面」重複很多次，用事件發生比例近似答案。

這種做法稱為 **Monte Carlo 模擬(Monte Carlo simulation)** 。每一次模擬都必須遵循題目指定的機率模型；模擬次數則控制數值近似的精細程度。

<a id="formula-ch09-monte-carlo-event-probability"></a>

### Monte Carlo 事件機率估計

令第 $b$ 次模擬的事件指標為

$$
I_b=
\begin{cases}
1, & \text{事件在第 }b\text{ 次模擬發生},\\
0, & \text{否則}.
\end{cases}
$$

執行 $B$ 次彼此獨立、且來自同一指定模型的模擬後，事件機率 $p$ 可近似為

$$
\hat p_{MC}=\frac{1}{B}\sum_{b=1}^{B}I_b.
$$

| 符號 | 意義 | 單位 |
|---|---|---|
| $p$ | 指定模型下真正的事件機率 | 無單位，介於 0 與 1 |
| $I_b$ | 第 $b$ 次模擬是否發生事件 | 0 或 1 |
| $B$ | 模擬重複次數 | 次 |
| $\hat p_{MC}$ | Monte Carlo 近似值 | 無單位，介於 0 與 1 |

**使用條件。** 每次模擬要忠實反映同一個目標模型；若要使用下一節的簡單誤差公式，各次模擬還要彼此獨立。

**何時使用。** 事件可模擬，但精確列舉或積分很麻煩時。

**何時不要使用。** 若未知的是母體分配，而你想評估樣本統計量的不確定性，只有反覆模擬某個任意指定模型並不夠；應先說明如何由資料估計該模型，這會導向 bootstrap。

### 1.2 大數法則保證的是模擬平均會穩定

**大數法則(Law of Large Numbers, LLN)** 指出：在適當條件下，獨立同分配且期望值存在的重複結果，其平均值會隨次數增加而趨近真實期望值。因為 $E(I_b)=p$，所以 $\hat p_{MC}$ 會隨 $B$ 增加而趨近 $p$。

這裡的關鍵是：LLN 讓你更精確地計算「你所模擬的模型」，卻不保證模型本身正確。若把公平硬幣寫成正面率 0.6，跑十億次只會非常精確地得到錯誤模型的答案。

<a id="formula-ch09-monte-carlo-standard-error"></a>

### Monte Carlo 機率估計的標準誤

獨立模擬下，$I_b$ 是成功機率為 $p$ 的 Bernoulli 隨機變數，因此

$$
SE_{MC}(\hat p_{MC})=\sqrt{\frac{p(1-p)}{B}}
\approx
\sqrt{\frac{\hat p_{MC}(1-\hat p_{MC})}{B}}.
$$

| 符號 | 意義 | 單位 |
|---|---|---|
| $SE_{MC}$ | 只由有限模擬次數造成的標準誤 | 與機率相同，無單位 |
| $p$ | 目標事件機率 | 無單位 |
| $\hat p_{MC}$ | 以模擬比例代入的機率估計 | 無單位 |
| $B$ | 獨立模擬次數 | 次 |

**量級檢查。** 標準誤隨 $1/\sqrt B$ 下降。要把 Monte Carlo 標準誤縮小為一半，約需把 $B$ 增為 4 倍。

**最壞情況檢查。** 因為 $p(1-p)\le 0.25$，所以 $SE_{MC}\le 1/(2\sqrt B)$。例如 $B=10{,}000$ 時不超過 0.005。

**常見混淆：兩層不確定性。** 若根據一份有限樣本做 bootstrap，會同時存在：

1. 原始樣本相對母體的**抽樣不確定性** ；
2. 只執行有限 $B$ 次重抽樣的 **Monte Carlo 誤差** 。

增加 $B$ 只能減少第 2 種，不能讓原始樣本更有代表性。

**非例。** 將同一組 12 位便利抽樣受訪者複製 10,000 次，不是 10,000 位獨立受訪者，也不會消除便利抽樣偏誤。

## 2. Plug-in principle：用可估計的世界代替未知世界

### 2.1 從抽樣分配到 bootstrap 分配

設未知母體分配為 $F$，我們關心母數

$$
\theta=t(F),
$$

而原始樣本 $x_1,\ldots,x_n$ 給出統計量

$$
\hat\theta=t(\hat F).
$$

問題是：$\hat\theta$ 若在重複抽樣中變動，它的分配長什麼樣？真實抽樣分配需要反覆從未知的 $F$ 抽樣，現實中做不到。

**Plug-in principle(代入原則)** 是用資料估計出的 $\hat F$ 代替未知的 $F$，再從 $\hat F$ 模擬重複抽樣。這個近似重複抽樣產生的 $\hat\theta^*$ 分配，稱為 **bootstrap 分配(bootstrap distribution)** 。

星號 $^*$ 一律表示「在 bootstrap 世界中計算的量」，不是乘法，也不是新的母數。

### 2.2 非參數 bootstrap：從經驗分配重抽

**經驗分配(empirical distribution)** $\hat F_n$ 對原始樣本中的每一筆觀測各放置機率 $1/n$。從 $\hat F_n$ 抽樣，等價於從原始的 $n$ 筆資料中 **有放回抽樣** 。

一次標準的非參數 bootstrap 流程是：

1. 原始資料有 $n$ 筆。
2. 從這 $n$ 筆中有放回抽取 $n$ 次，形成一份 bootstrap 樣本 $x_1^*,\ldots,x_n^*$。
3. 用與原始分析完全相同的規則，算出 $\hat\theta^*=s(x_1^*,\ldots,x_n^*)$。
4. 重複步驟 2–3 共 $B$ 次，得到 $\hat\theta^{*(1)},\ldots,\hat\theta^{*(B)}$。

**為什麼必須有放回？** 有放回時，同一筆觀測可出現零次、一次或多次，bootstrap 樣本才會彼此不同。若從 $n$ 筆中無放回抽 $n$ 筆，每次都只是原資料的重新排序，多數統計量完全不變。

**為什麼每次仍抽 $n$ 筆？** 我們要近似「樣本大小為 $n$ 時」統計量的抽樣分配。改成抽 $2n$ 筆，估計的是另一個樣本大小下的變動程度。

**例子。** 原始等待時間為 $[2,4,5,9]$ 分鐘。合法的 bootstrap 樣本可能是 $[2,2,5,9]$，也可能是 $[4,4,4,5]$。每份都抽 4 筆且允許重複。$[2,4,5]$ 大小不對；$[2,4,5,9]$ 的隨機排列若來自無放回抽樣，也沒有模擬經驗分配下的獨立抽取。

### 2.3 非參數不等於沒有假設

非參數 bootstrap 不指定常態或其他參數族，但仍隱含重要條件：

- 原始觀測應能代表目標母體。
- 標準逐筆重抽通常假設觀測為獨立同分配(independent and identically distributed, i.i.d.)。
- 樣本要充分呈現影響統計量的分配特徵。
- 統計量對分配的小變化不應極度不規則。

時間序列、同一人的重複測量、同班學生或同醫院病人通常有相依性，不能直接把每一列當成可交換的獨立單位。這類資料可能需要 block bootstrap 或 cluster bootstrap；這些方法是延伸內容，但選擇重抽單位的原則相同：**重抽樣必須保留原設計中的依賴結構。**

**非例。** 對 20 對雙胞胎的 40 列資料逐人重抽，會拆散配對關係；若分析目標依賴成對差異，應以「一對」或「成對差值」為重抽單位。

## 3. 用 bootstrap 分配估計標準誤與偏誤

假設已取得 $B$ 個 bootstrap 複本(replicates)$\hat\theta^{*(1)},\ldots,\hat\theta^{*(B)}$。它們的散布近似原統計量 $\hat\theta$ 在重複抽樣下的散布。

<a id="formula-ch09-bootstrap-standard-error"></a>

### Bootstrap 標準誤

先計算 bootstrap 複本平均：

$$
\bar{\hat\theta}^{*}=\frac{1}{B}\sum_{b=1}^{B}\hat\theta^{*(b)}.
$$

bootstrap 標準誤為複本的樣本標準差：

$$
\widehat{SE}_{boot}(\hat\theta)
=
\sqrt{
\frac{1}{B-1}
\sum_{b=1}^{B}
\left(\hat\theta^{*(b)}-\bar{\hat\theta}^{*}\right)^2
}.
$$

| 符號 | 意義 | 單位 |
|---|---|---|
| $\hat\theta$ | 原始樣本算出的統計量 | 與目標量相同 |
| $\hat\theta^{*(b)}$ | 第 $b$ 份 bootstrap 樣本的統計量 | 與 $\hat\theta$ 相同 |
| $\bar{\hat\theta}^{*}$ | $B$ 個 bootstrap 複本的平均 | 與 $\hat\theta$ 相同 |
| $\widehat{SE}_{boot}$ | bootstrap 估計的抽樣標準誤 | 與 $\hat\theta$ 相同 |
| $B$ | bootstrap 重複次數 | 次 |

**使用條件。** 重抽樣機制要合理近似原本的資料生成與抽樣機制，且 $B$ 要足以讓估計在重跑時穩定。

**單位檢查。** 若 $\hat\theta$ 是分鐘，bootstrap 標準誤也是分鐘；若是迴歸斜率「萬元／年」，標準誤也必須是萬元／年。

**何時使用。** 統計量的解析標準誤未知、推導困難，或想用模擬檢查近似公式時。

**何時不要直接使用。** 樣本極小、尾端或最大值等特徵在樣本中幾乎沒有資訊、觀測相依卻逐列重抽，或統計量位於參數空間邊界時，普通 bootstrap 可能失真。

<a id="formula-ch09-bootstrap-bias"></a>

### Bootstrap 偏誤估計

若估計量的真實偏誤是 $E_F(\hat\theta)-\theta$，plug-in 近似得到

$$
\widehat{bias}_{boot}
=
\bar{\hat\theta}^{*}-\hat\theta.
$$

| 符號 | 意義 | 單位 |
|---|---|---|
| $\bar{\hat\theta}^{*}$ | bootstrap 複本平均 | 與 $\theta$ 相同 |
| $\hat\theta$ | 原始資料估計值；在 bootstrap 世界扮演母數角色 | 與 $\theta$ 相同 |
| $\widehat{bias}_{boot}$ | 估計的偏誤方向與大小 | 與 $\theta$ 相同 |

正值表示 bootstrap 世界中的估計量平均高於 plug-in 目標；負值表示平均偏低。

可寫出偏誤修正估計 $\hat\theta_{bc}=\hat\theta-\widehat{bias}_{boot}$，但**不應看到非零偏誤就自動修正** 。偏誤估計本身也有變動，修正可能增加均方誤差，且無法修復抽樣設計偏誤。

**常見混淆。** bootstrap 標準誤描述複本的散布；bootstrap 偏誤比較複本中心與原估計值。兩者不是同一件事。

1. bootstrap 偏誤估計是多少？
2. bootstrap 標準誤是多少？
3. 兩者的單位是否與原統計量相同？

</details>

## 4. 參數式 bootstrap：從估計出的模型抽樣

非參數 bootstrap 把經驗分配當作 $\hat F$。另一條路是先假設母體屬於某個參數模型 $F_\eta$，由資料估計參數 $\hat\eta$，再從 $F_{\hat\eta}$ 產生樣本。這稱為 **參數式 bootstrap(parametric bootstrap)** 。

流程如下：

1. 指定資料模型 $F_\eta$，並檢查它對情境是否合理。
2. 用原始資料估計 $\hat\eta$。
3. 從 $F_{\hat\eta}$ 產生大小為 $n$ 的 bootstrap 樣本。
4. 每次重新執行完整估計流程，得到 $\hat\theta^*$。
5. 重複 $B$ 次，摘要其標準誤、偏誤或分位數。

例如假設等待時間為指數分配，先由資料估計速率 $\hat\lambda$，再從 $Exponential(\hat\lambda)$ 產生每份大小為 $n$ 的資料。這會生成原樣本未出現過的數值；非參數 bootstrap 則只會重複原觀測值。

| 方法 | plug-in 分配 | 優點 | 主要風險 |
|---|---|---|---|
| 非參數 bootstrap | 經驗分配 $\hat F_n$ | 少指定分配形狀 | 小樣本可能沒呈現尾端；仍需正確處理相依性 |
| 參數式 bootstrap | 擬合模型 $F_{\hat\eta}$ | 合理模型下可平滑地模擬未觀測值 | 模型形狀錯誤會系統性扭曲結果 |

**選擇提示。** 若科學機制支持特定模型，而且模型診斷合理，參數式 bootstrap 可能更有效率。若不願強加分配形狀，且樣本足以代表分配，非參數方法較自然。

**非例。** 資料明顯右偏且有長尾，只因常態分配容易模擬就使用常態參數式 bootstrap。大量重複不能補救錯誤的尾端模型。

<details>
<summary>核對答案</summary>

是 $\hat F$：非參數方法使用經驗分配；參數式方法使用以資料估計參數後的模型。兩者通常都維持原樣本大小 $n$，也都需要足夠的 $B$。

</details>

## 5. Bootstrap 信賴區間

bootstrap 分配不只提供標準誤，也可直接用它的分位數建立信賴區間。本章先使用最直觀的 **percentile bootstrap interval(百分位 bootstrap 信賴區間)** 。

<a id="formula-ch09-bootstrap-percentile-interval"></a>

### Percentile bootstrap 信賴區間

令 $q^*_{\gamma}$ 表示 $B$ 個 bootstrap 複本的第 $\gamma$ 分位數。名目信賴水準為 $100(1-\alpha)\%$ 的 percentile interval 是

$$
CI_{perc}=
\left[
q^*_{\alpha/2},
q^*_{1-\alpha/2}
\right].
$$

95% 區間使用 $\alpha=0.05$，即 bootstrap 複本的第 2.5 與第 97.5 百分位數。

| 符號 | 意義 | 單位 |
|---|---|---|
| $\alpha$ | 區間兩端合計的名目尾端比例 | 無單位 |
| $q^*_{\gamma}$ | bootstrap 複本的第 $\gamma$ 分位數 | 與 $\theta$ 相同 |
| $CI_{perc}$ | percentile bootstrap 信賴區間 | 兩端點皆與 $\theta$ 同單位 |

**直覺。** 若 bootstrap 世界合理近似真實世界，而且估計量的誤差結構適合 percentile 方法，bootstrap 複本中央 95% 的範圍可用來校準未知母數的合理區間。

**解讀。** 在相同抽樣設計與建構程序長期重複下，名目 95% 的程序希望約有 95% 的區間涵蓋固定母數。對已算出的單一區間，不應說「母數有 95% 機率落在其中」。

**何時使用。** 想容許區間不對稱，而且 bootstrap 分配能合理反映估計量的形狀時。

**何時不要把它當萬用答案。** 強烈偏誤、小樣本、極端分位數、最大值、離散統計量、邊界問題或不規則估計量，percentile interval 的覆蓋率可能很差。更進階的 basic、studentized 或 BCa 區間有不同校正，但也各有條件；不可只挑看起來最順眼的一種。

**模擬次數提示。** 分位數使用分配尾端，通常比只估標準誤更需要較大的 $B$。應固定隨機種子以便重現，並用不同種子重跑，確認端點對 Monte Carlo 變動不敏感。沒有一個 $B$ 能保證所有問題都足夠。

**非例。** 95% bootstrap 區間不含 0，不能單憑這點斷言效果很重要或有因果關係。實質重要性與研究設計仍需另行判斷。

1. 95% percentile interval 是什麼？
2. 可以說真實斜率有 95% 機率在這個已實現區間內嗎？

## 6. 迴歸中的 bootstrap：究竟要重抽什麼？

考慮簡單線性迴歸

$$
Y_i=\beta_0+\beta_1x_i+\varepsilon_i.
$$

想估計斜率 $\hat\beta_1$ 的不確定性時，不能只說「做 bootstrap」；必須先決定 $x$ 是隨機取得還是固定設計、誤差結構如何，以及哪個觀測單位彼此獨立。

### 6.1 Pairs bootstrap：重抽整對 $(x_i,y_i)$

**成對 bootstrap(pairs bootstrap)** 把每一列 $(x_i,y_i)$ 視為一個抽樣單位：

1. 從原本 $n$ 對 $(x_i,y_i)$ 中有放回抽 $n$ 對。
2. 對每份 bootstrap 資料重新擬合迴歸。
3. 儲存 $\hat\beta_1^*$ 或其他目標統計量。

它近似的是成對資料由某個聯合分配抽出的情況，因此讓 bootstrap 樣本中的 $x$ 也會改變。它不要求先指定誤差的參數分配，但仍假設各資料對是可獨立重抽的代表性單位。

**注意。** 某些 bootstrap 樣本可能幾乎沒有 $x$ 變異，使斜率很不穩；這是原資料在設計或樣本量上的警訊，不能悄悄刪除後當作沒發生。

### 6.2 Residual bootstrap：固定 $x$，重抽誤差

若 $x_i$ 是研究者固定的設計點，且線性模型與同質、獨立誤差的假設合理，可使用 **殘差 bootstrap(residual bootstrap)** ：

1. 擬合原模型，取得 $\hat y_i=\hat\beta_0+\hat\beta_1x_i$ 與殘差 $e_i=y_i-\hat y_i$。
2. 通常先將殘差中心化：$e_i^c=e_i-\bar e$。
3. 有放回抽取殘差 $e_i^*$，並建立 $y_i^*=\hat y_i+e_i^*$，保留原本的 $x_i$。
4. 對 $(x_i,y_i^*)$ 重新擬合，儲存 $\hat\beta_1^*$。

<a id="formula-ch09-residual-bootstrap-response"></a>

### Residual bootstrap 的模擬反應值

$$
y_i^*=\hat y_i+e_i^*
=\hat\beta_0+\hat\beta_1x_i+e_i^*.
$$

| 符號 | 意義 | 單位 |
|---|---|---|
| $x_i$ | 固定保留的第 $i$ 個解釋變數 | $x$ 的單位 |
| $\hat y_i$ | 原模型在 $x_i$ 的 fitted value | $y$ 的單位 |
| $e_i^*$ | 從中心化殘差重抽的誤差代理 | $y$ 的單位 |
| $y_i^*$ | bootstrap 世界的模擬反應值 | $y$ 的單位 |

**單位檢查。** $\hat y_i$ 與 $e_i^*$ 都是 $y$ 的單位，因此可相加。

**使用條件。** 線性平均結構合理；誤差彼此獨立、同分配且變異大致相同；固定 $x$ 的觀點符合研究設計。若異質變異明顯，直接把所有殘差混在一起重抽會把大小不同的誤差錯誤交換。

### 6.3 方法選擇不是軟體選項，而是科學假設

| 問題線索 | 較自然的起點 | 原因 |
|---|---|---|
| $(X,Y)$ 是從目標族群隨機觀察到的資料對 | pairs bootstrap | 同時反映 $X$ 與 $Y$ 的抽樣變動 |
| 實驗者固定各個 $x_i$，且同質誤差模型合理 | residual bootstrap | 保持設計點不動，只模擬條件誤差 |
| 同一受試者有多列資料 | 兩者皆不能直接逐列套用 | 應保留受試者內相依性，例如以受試者為重抽單位 |
| 殘差散布隨 $x$ 改變 | 普通 residual bootstrap 不合適 | 混抽殘差破壞條件變異結構 |

**非例。** 隨機抽到 100 家公司的資料，卻做 residual bootstrap 並把這 100 個公司的 $x$ 當作研究者固定設計；這會回答條件於現有 $x$ 的問題，未必符合要推論公司族群的目標。

## 7. 重抽樣的診斷與限制

### 7.1 三個模型層次都要檢查

做完漂亮的 bootstrap 直方圖前，依序問：

1. **資料層：** 原始樣本是否代表目標母體？是否有選擇偏誤、遺漏值或測量問題？
2. **重抽層：** 重抽單位是否正確？有沒有保留分層、群聚、配對或時間相依？
3. **統計量層：** 目標統計量在目前樣本量下是否穩定？某一筆資料是否主宰結果？

bootstrap 主要處理在既定資料與模型下的隨機變動，不能自動處理所有系統性誤差。

### 7.2 常見失效警訊

- **樣本太小或稀有類別未出現：** 經驗分配無法重抽出沒見過的重要類型。
- **估計極端值：** 樣本最大值的非參數 bootstrap 永遠不會超過原樣本最大值，難以模擬未知尾端。
- **離群值與高影響點：** 某筆資料在複本中重複多次或完全缺席，可能形成多峰或極長尾；這既可能揭露真實敏感度，也可能暴露資料品質問題。
- **相依資料逐筆重抽：** 會錯估有效資訊量。
- **參數位於邊界：** 例如變異成分接近 0，普通近似可能不規則。
- **先看資料再選分析：** 若每份 bootstrap 沒有重做同一套選模流程，會低估整個分析程序的不確定性。

### 7.3 最低限度的穩健檢查

- 固定並記錄隨機種子，使結果可重現。
- 用另一個種子重跑，檢查標準誤與區間端點是否穩定。
- 畫出 bootstrap 分配，檢查偏斜、多峰、極端值與失敗擬合。
- 比較合理的解析近似或不同 bootstrap 方案；差異大時，先查假設，不要只選喜歡的答案。
- 報告 $B$、重抽單位、是否參數式、統計量定義及區間方法。
- 若資料來自複雜抽樣設計，讓重抽樣反映權重、分層與群聚，而不是套用普通 i.i.d. bootstrap。

**重點提醒。** 「程式成功跑完」只證明演算法有輸出，不證明推論有效。

## 章末整合：先辨認你在模擬哪一個世界

遇到重抽樣問題，可沿著以下順序判斷：

1. **目標是什麼？** 機率、標準誤、偏誤、信賴區間，還是迴歸係數的不確定性？
2. **未知的是什麼？** 若機率模型已指定，通常是 Monte Carlo 數值近似；若母體分配未知且由樣本代入，通常是 bootstrap。
3. **plug-in 分配是什麼？** 經驗分配，還是擬合後的參數模型？
4. **獨立抽樣單位是什麼？** 人、配對、家庭、公司、時間區塊，還是迴歸資料對？
5. **每份樣本要保留什麼結構？** 樣本大小、配對、群聚、固定設計點或異質變異。
6. **要摘要什麼？** 複本標準差、偏誤、分位數，或整個分配形狀。
7. **結果穩定且假設合理嗎？** 增加 $B$ 只處理模擬誤差，不能代替這項檢查。

## 公式與術語速查

| 名稱 | 核心內容 | 決策提示 |
|---|---|---|
| Monte Carlo 機率估計 | $\hat p_{MC}=B^{-1}\sum I_b$ | 指定模型下，以事件比例近似機率 |
| Monte Carlo 標準誤 | $\sqrt{\hat p_{MC}(1-\hat p_{MC})/B}$ | 衡量有限模擬次數造成的數值誤差 |
| 大數法則 | 模擬平均隨次數增加而穩定 | 不會修正錯誤模型或抽樣偏誤 |
| Plug-in principle | 用 $\hat F$ 代替未知 $F$ | bootstrap 的建模核心 |
| 非參數 bootstrap | 從原樣本有放回抽 $n$ 筆 | 使用經驗分配，通常需 i.i.d. 或相應調整 |
| 參數式 bootstrap | 從 $F_{\hat\eta}$ 產生資料 | 模型合理時使用，須承擔模型錯置風險 |
| Bootstrap 標準誤 | bootstrap 複本的樣本標準差 | 單位與目標統計量相同 |
| Bootstrap 偏誤 | $\bar{\hat\theta}^*-\hat\theta$ | 比較複本中心與原估計值 |
| Percentile interval | $[q^*_{\alpha/2},q^*_{1-\alpha/2}]$ | 直接取複本分位數，需注意覆蓋限制 |
| Pairs bootstrap | 重抽整對 $(x_i,y_i)$ | 隨機資料對、希望反映 $X$ 的變動 |
| Residual bootstrap | $y_i^*=\hat y_i+e_i^*$ | 固定 $x$ 且同質獨立誤差合理 |

2. 近似樣本大小 20 的變動，而不是原研究的 25。

## 練習與逐步詳解

以下題目不是把數字換掉後反覆套公式。每題先判斷「模擬哪個世界」與「重抽哪個單位」，再進行計算。

### 題 9-1：增加模擬次數能解決什麼？

**題目 ID：** `m09-monte-carlo-error-01`
**類型：** 概念選擇題
**難度：** 基礎

研究者從 30 位自願填答者取得資料，以非參數 bootstrap 重抽 2,000 次。將重抽次數增加到 200,000 次，最直接能改善哪一件事？

A. 消除自願填答造成的選擇偏誤
B. 降低有限 bootstrap 次數造成的 Monte Carlo 誤差
C. 讓 30 人的原始樣本等同於 200,000 位獨立受訪者
D. 保證 percentile interval 的實際覆蓋率恰好為 95%

<details>
<summary>答案、選項診斷與詳解</summary>

**答案：B。** 增加 $B$ 會讓同一 bootstrap 世界的數值摘要更穩定，但不會增加原始樣本的資訊。

- **A 的迷思：把計算誤差當成抽樣設計偏誤。** bootstrap 會反覆帶入原始樣本的選擇機制；應改善抽樣設計，而不是只增加 $B$。
- **C 的迷思：把重複使用資料當成新增獨立觀測。** 200,000 份複本都源自相同 30 人，沒有新增受訪者。
- **D 的迷思：把名目信心水準當成有限樣本保證。** 覆蓋率還取決於樣本、統計量、重抽機制與區間方法。

這題連結 [Monte Carlo 標準誤](#formula-ch09-monte-carlo-standard-error)。由 $SE_{MC}\propto1/\sqrt B$ 可知，增加模擬次數只直接處理數值模擬的變動。

</details>

### 題 9-2：已知模型還是未知母體？

**題目 ID：** `m09-method-world-01`
**類型：** 方法選擇題
**難度：** 基礎

下列哪個問題最直接屬於 Monte Carlo 模擬，而不是 bootstrap？

A. 已知某遊戲每局獲勝率為 0.4，模擬 100 局後總獲勝數至少為 50 的機率
B. 用 80 筆實際交易時間，有放回重抽以估計樣本中位數的標準誤
C. 由樣本估計常態模型參數，再重複產生資料以估計估計量偏誤
D. 從觀測到的 $(x_i,y_i)$ 資料對重抽，估計迴歸斜率的不確定性

<details>
<summary>答案、選項診斷與詳解</summary>

**答案：A。** 機率模型與參數已由題目指定，模擬是在近似該模型下的事件機率。

- **B 的迷思：只要使用電腦反覆抽樣就都叫 Monte Carlo。** 這裡用經驗分配代替未知母體，是非參數 bootstrap。
- **C 的迷思：從參數模型抽樣就一定不是 bootstrap。** 參數由原樣本估計後再模擬，正是參數式 bootstrap。
- **D 的迷思：迴歸重抽樣是獨立於 bootstrap 的第三類方法。** 重抽資料對是 pairs bootstrap。

判斷關鍵不是軟體或重複次數，而是模擬世界從何而來：A 的模型已知；B–D 都以原始資料建立 plug-in 世界。

</details>

### 題 9-3：計算 Monte Carlo 標準誤

**題目 ID：** `m09-monte-carlo-se-01`
**類型：** 計算題
**難度：** 基礎

在 $B=6{,}400$ 次獨立模擬中，事件發生比例為 $\hat p_{MC}=0.36$。估計這個機率近似值的 Monte Carlo 標準誤。

<details>
<summary>逐步詳解</summary>

1. **辨認已知量。** $B=6{,}400$ 次，$\hat p_{MC}=0.36$，兩者皆無測量單位。
2. **選擇方法。** 目標是有限獨立模擬造成的數值誤差，使用 [Monte Carlo 標準誤](#formula-ch09-monte-carlo-standard-error)，不是 bootstrap 複本標準差。
3. **檢查條件。** 題目說明各次模擬獨立，且事件指標來自同一模型，因此可使用 Bernoulli 比例近似。
4. **代入公式。** $\widehat{SE}_{MC}=\sqrt{0.36(1-0.36)/6{,}400}$。
5. **計算。** $0.36\times0.64=0.2304$，$0.2304/6{,}400=0.000036$，所以 $\widehat{SE}_{MC}=\sqrt{0.000036}=0.006$。
6. **情境解讀。** 若重跑整套 6,400 次模擬，$\hat p_{MC}$ 只因有限模擬次數造成的典型變動約為 0.006，也就是 0.6 個百分點。
7. **合理性檢查。** 最壞情況上界為 $1/(2\sqrt{6{,}400})=0.00625$；答案 0.006 小於上界，量級合理。

**答案：** $\widehat{SE}_{MC}=0.006$，或 0.6 個百分點。

</details>

### 題 9-4：同時計算 bootstrap 標準誤與偏誤

**題目 ID：** `m09-bootstrap-se-bias-01`
**類型：** 計算與解讀題
**難度：** 中等

為了示範手算，某研究只列出 $B=5$ 個 bootstrap 複本：$8,10,11,13,13$ 秒。原始樣本的統計量為 $\hat\theta=10.5$ 秒。計算 bootstrap 標準誤與 bootstrap 偏誤估計。(實務上 $B=5$ 遠遠不足。)

<details>
<summary>逐步詳解</summary>

1. **辨認已知量。** 五個複本為 $8,10,11,13,13$ 秒，$B=5$，原估計為 10.5 秒。
2. **選擇方法。** 複本的樣本標準差估計標準誤，連結 [bootstrap 標準誤](#formula-ch09-bootstrap-standard-error)；複本平均減原估計值估計偏誤，連結 [bootstrap 偏誤](#formula-ch09-bootstrap-bias)。
3. **檢查條件。** 題目只供手算，無法由五個摘要值驗證原重抽機制；真正分析仍須確認重抽單位與 plug-in 模型合理。
4. **代入標準誤公式。** 複本平均為 $\bar{\hat\theta}^*=(8+10+11+13+13)/5=11$ 秒。平方離差和為 $(-3)^2+(-1)^2+0^2+2^2+2^2=18$ 秒平方。
5. **計算標準誤與偏誤。** $\widehat{SE}_{boot}=\sqrt{18/(5-1)}=\sqrt{4.5}\approx2.12$ 秒；$\widehat{bias}_{boot}=11-10.5=0.5$ 秒。
6. **情境解讀。** 此 bootstrap 世界估計原統計量的抽樣標準誤約 2.12 秒，並顯示估計量平均比 plug-in 目標高約 0.5 秒。
7. **合理性檢查。** 標準誤與偏誤都保留「秒」的單位；標準誤不可能是秒平方。複本範圍為 5 秒，2.12 秒的散布量級合理。

**答案：** bootstrap 標準誤約 2.12 秒；bootstrap 偏誤估計為 $+0.5$ 秒。

</details>

### 題 9-5：Percentile interval 的解讀

**題目 ID：** `m09-percentile-interpretation-01`
**類型：** 解讀選擇題
**難度：** 中等

某研究以相同程序得到母體中位數的 95% percentile bootstrap interval 為 $[42,58]$ 分鐘。哪個解讀最恰當？

A. 固定的母體中位數有 95% 機率位於這個已算出的區間
B. 約 95% 的個別觀測值位於 42 到 58 分鐘
C. 若在相同條件下反覆抽樣並用同一程序建區間，長期希望約 95% 的區間涵蓋母體中位數
D. 未來一筆觀測有 95% 機率位於 42 到 58 分鐘

<details>
<summary>答案、選項診斷與詳解</summary>

**答案：C。** 信心水準描述程序的長期覆蓋率，不是已實現區間對固定母數的機率敘述。

- **A 的迷思：將頻率學派信賴區間當成母數的後驗機率區間。** 區間算出後，固定母數不被視為隨機變數。
- **B 的迷思：混淆母數區間與資料中央範圍。** 這是母體中位數的不確定性區間，不是個別資料的 95% 範圍。
- **D 的迷思：混淆信賴區間與預測區間。** 預測新觀測還要納入個體層級變動，通常更寬。

計算結構連結 [percentile bootstrap 信賴區間](#formula-ch09-bootstrap-percentile-interval)。即使程序名目為 95%，實際覆蓋率仍取決於方法條件是否合理。

</details>

### 題 9-6：非參數還是參數式 bootstrap？

**題目 ID：** `m09-bootstrap-type-01`
**類型：** 方法選擇題
**難度：** 中等

工程師有 120 個元件壽命。物理機制與診斷都支持 Weibull 模型，目標是估計一個複雜可靠度統計量的標準誤，並希望模擬出原樣本未出現過但模型允許的壽命。下列哪個起點最合理？

A. 參數式 bootstrap：估計 Weibull 參數，再從擬合模型產生大小 120 的樣本
B. 非參數 bootstrap：無放回抽取全部 120 筆
C. 將原始資料複製很多次，當作新增的獨立元件
D. 隨意選常態模型做參數式 bootstrap，因為常態亂數容易產生

<details>
<summary>答案、選項診斷與詳解</summary>

**答案：A。** 題目提供實質模型依據與診斷支持；參數式 bootstrap 可從 $F_{\hat\eta}$ 產生模型允許的新數值。

- **B 的迷思：忽略有放回抽樣的必要性。** 無放回抽全部資料只會重新排序，統計量通常不變；若選非參數方法，應有放回抽 120 筆。
- **C 的迷思：把資料複製當成新資訊。** 複製不會新增獨立元件，也不會正確產生抽樣分配。
- **D 的迷思：把計算便利當成模型證據。** 參數式 bootstrap 的有效性依賴模型合理，而不是亂數產生器是否方便。

若 Weibull 假設不受支持，A 的優勢就會消失；方法選擇必須隨證據改變。

</details>

### 題 9-7：迴歸要重抽資料對還是殘差？

**題目 ID：** `m09-regression-bootstrap-01`
**類型：** 方法選擇題
**難度：** 中等

農業實驗者事先固定四個施肥量，在每個施肥量下獨立量測產量。線性關係合理，殘差變異大致一致。研究者要在保留這四個設計點的條件下估計斜率標準誤。哪個方法最貼近設計？

A. residual bootstrap，固定施肥量並令 $y_i^*=\hat y_i+e_i^*$
B. pairs bootstrap，因為任何迴歸都必須讓 $x$ 一起重抽
C. 只重抽 $y_i$，再任意配回固定的 $x_i$
D. 只增加 $B$，不必檢查線性與同質變異

<details>
<summary>答案、選項診斷與詳解</summary>

**答案：A。** $x$ 是固定設計點，且普通 residual bootstrap 的誤差條件在題目中大致成立。

- **B 的迷思：認為 pairs bootstrap 永遠比 residual bootstrap 一般。** pairs bootstrap 會改變 bootstrap 樣本中的設計點比例，與題目要固定 $x$ 的條件不一致。
- **C 的迷思：把反應值視為可脫離條件平均結構交換。** 任意重配會破壞 $Y$ 隨 $X$ 改變的關係。
- **D 的迷思：以模擬次數取代模型診斷。** 大 $B$ 不能修正非線性或異質變異造成的錯誤重抽機制。

模擬反應值的公式連結 [residual bootstrap](#formula-ch09-residual-bootstrap-response)。若殘差變異隨施肥量改變，普通 residual bootstrap 就不再是自然選擇。

</details>

### 題 9-8：相依資料的重抽單位

**題目 ID：** `m09-resampling-unit-01`
**類型：** 設計診斷題
**難度：** 挑戰

研究者抽取 25 所學校，每校調查 20 位學生。學生在同校內可能相似。若要估計整體平均讀書時間的標準誤，下列哪個做法最值得優先考慮？

A. 把 500 位學生當成 i.i.d. 個體逐人重抽
B. 以學校為群聚重抽單位，並在分析中保留校內資料結構
C. 每校只留下第一位學生，再逐人重抽
D. 逐人重抽，但把 $B$ 設為一百萬就能補償校內相依

<details>
<summary>答案、選項診斷與詳解</summary>

**答案：B。** 抽樣設計的獨立單位主要是學校；重抽整個群聚可保留校內相依。實際方法還要配合原抽樣設計與加權方式。

- **A 的迷思：把資料列數等同於獨立資訊量。** 同校學生不是 500 個完全獨立的抽樣單位，逐人重抽通常會低估標準誤。
- **C 的迷思：為取得獨立性而任意丟棄多數資料。** 這會浪費校內資訊，也改變原本的估計程序。
- **D 的迷思：認為極大 $B$ 可修正錯誤依賴模型。** 增加 $B$ 只能穩定錯誤重抽機制下的答案。

**方法理由：** bootstrap 必須模仿原資料如何取得。若第一階段抽的是學校，就不能默認每位學生都可獨立交換。這題不要求計算；真正實作時還需確認學校抽樣是否等機率、每校樣本數及權重。

</details>
